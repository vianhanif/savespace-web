# Save Space Frontend

## Installation

Make sure you have nodejs v.8*+.
[Install Nodejs](https://www.nodejs.org)

#### Install yarn using npm<br/>
```bash
npm i -g yarn
```

#### Install Packages<br/>
```bash
yarn
```

#### Run Webpack<br />
```bash
yarn start
```

#### Run unit Test<br />
```bash
yarn test
```

#### Run Prettier<br />
```bash
yarn prettier
```

#### Watch Prettier<br />
```bash
yarn prettier-watch
```

#### Watch Flow<br />
```bash
yarn flow:watch
```

#### Staging Build <br/>
```bash
yarn build:staging
```

#### Production Build <br/>
```bash
yarn build
```
